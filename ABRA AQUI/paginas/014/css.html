css internos/ head

esse formato já se encaixa quando tem alguma modificação somente na quela pagina do site.

na parte do head da pagina html , apos o title colocar a tag <style>.

Dentro da tag style vai colocar qual tag do html gostaria de modificar , como mostrado a seguir:

          <style>

                selector { 
                   property: value;
                 }

            </style>

apos colocar a tag que gostaria de modificar, sempre abrir e fechar chaves. Tambem é importante resaltar que apos finalizar o comando de modificação em css, smepre colocar um ; no final de tudo , para mostrar que esta encerrando aquele modificao.


css externo

agora se a modificação que vc ira fazer em uma pagina sera em todas, é mais usual e facil utilizar o css externo a sua pagina do html.

utilizando a tag link :

<link rel="stylesheet" href="estilo.css">

dessa forma o program vai chamar uma pagina que tera as alterações em css, não presisando colocar as modificações na pagina, mas sim em uma externa.

na pagina do css é importante colocar a regra de @charset "utf-8"; ,pois essa regra fara com que o programa nao tenha problema com a gramatica brasileira.

obs: ordem de comando

no html ele sempre vai obdecer a tag que estara inline (dentro da propria tag), interna (na parte da head), e por ultimo o externo (em uma ppagina externa propia com css)

         

        tag tag(tem que estar dentro da primeira trag){ 
           property: value;
         }

para classes

quando vc determina class="" dentro de uma tag no html , aquela tag tera suas modificações de acordo com o nome da class e não mais pela tag.

         

        .nomeDaCLasse{ 
           property: value;
         }

se colocar uma tag.nomeDaClasse as modificações só serão validas para a class que tiver dentro da tag dita antes do .

da para uma tag ter duas class, dentro da tag class=" nomeDaCLasse nomeDaCLasse", assim mesmo que sejam duas class separadas pelo spaço aquela tag tera as alterações das duas.

obs: sempre tem nque por o ponto grudado no nome da class , para indicar que é uma class.


ID

o id funciona bem parecido com o class, porem na hora do css, para chamar um id se coloca a # e depois o nome do id.
Mesma coisa da class só troca o . por #.

Porem ele tem suas diferenças, como uma tag não pode ter mais de um id, como no class pode. O id não pode ser repetido, ou seja enquanto no class vc poderia usar uma mesma class par outras tags o id não pode, é somente para uma.
Ele tambêm é inportante para o javascript encontrar elementos únicos na página, como tambem pode ser usado na URL para rolar até certo elemento da página


DIV

Essa tag divide o programa em blocos para as funçoes.
É uma tag que não sera mostrada ao usuario , é somente uma tag de organização.

Essa tag facilita colocar um estilo atravez do css, em um bloco inteiro , é mais rapido e organizado do que atravez de uma id ou class em cada tag.

A tag div normalmente vem acompanhada do id , para que um bloco do seu html seja modificado de uma vez.

<div> conteudo , como outras tgas </div>
