caixa de texto

A caixa de texto é determinada na tag type=" text" . para funcionar o todo presisa que a tag form,da tag label e input juntas:

<form>

<label for ="o nome da variavel aonde será armazenada" > texto de instrução ao usuario </label>

<input type="text" name="caixa1" value="mostra uma mensagem na caixa de resposta">

</form>

obs: a tag name é a mesma da taga for, ela é a tag que armazenera a resposta do usuario.
