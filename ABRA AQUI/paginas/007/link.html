links internos

links internos são links que levaram para outra pagina , mas do seu proprio site.

os comando não mudam , mas sim o jeito que você vai usar eles.

utiliza a estrutura de codigo a seguir, para fazer links internos de arquino.html que estejam na mesma pasta :

<a href="nome do arquivo .html" rel="next" target="_self"(esta indicando para o google que esta direcionadno para uma pagina sua mesmo)> A frase/ palavra a ser transformada em link </a>
